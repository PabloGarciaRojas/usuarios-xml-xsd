**Actividad Evaluativa: Validación de XML con XSD y Expresiones Regulares**



2 archivos:

* usuarios.xsd: es el archivo que define las reglas del XML
* usuarios.xml: es el archivo que contiene las pruebas para la comprobación de las restricciones del xsd.



Restricciones del xsd:

* Nombre de usuario: tiene como mínimo 5 y como máximo 15 caracteres, el primero empieza por minúscula.
* Email: utiliza el formato estándar de correo que es nombre@dominio.extension
* Teléfono: tiene 9 numero divididos en 3 bloques ej: 691 275 401
* Código Postal: solamente pueden ser aceptados los de España.
* Contraseña: tiene como mínimo 8 caracteres, tiene letras, numero y símbolos permitidos.
